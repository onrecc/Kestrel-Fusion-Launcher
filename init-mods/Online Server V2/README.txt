You are responsible of your server.
The public-rules.txt is a file where you can write the rules (using the UTF-8 encoding) that the peoples will see if they try to join your server.
To open and edit this public-rules.txt file, go to "{MODPATH}", right click on the file named public-rules and open it with the notepad.
This public-rules.txt file must have less than 10000 characters else the peoples that want to join your server can't see the rules.
Before activating this mod, always check that the public-rules.txt contains your rules because when you update the application these rules may be deleted.
If you don't know what you do or if you don't know what this mod do, disable this mod.
